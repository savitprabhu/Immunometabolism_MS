
Directory: /input

This folder contains all the raw data necessary for analysis:
1. Directory: immgen_rawdata contains microarray data downloaded from immgen
2. Directory: GEO2R_downloads contains results of differential expression analysis done using GEO2R tool
3. Directory: curated_genelist contains curated list of pathways and genes
4. Directory: Aif_correlations contains raw data of correlations of various genelists with Aif, downloaded from immuno-navigator website
